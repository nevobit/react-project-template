## Real Vision React Template

Esta es una plantilla de proyecto para aplicaciones de React.

Para crear un nuevo proyecto basa en esta plantilla use:

### Get started

Install the dependenciers

```
    cd react-project-template
    npm install

```

Run project

```
    npm start

```
